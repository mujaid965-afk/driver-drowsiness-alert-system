
# Driver Drowsiness Alert System

This is a mini project developed as part of my Electronics and Communication Engineering curriculum.

## Project Description
The system detects driver drowsiness by monitoring eye-blink patterns using a camera.
When signs of fatigue are detected, an alert is triggered to warn the driver.
